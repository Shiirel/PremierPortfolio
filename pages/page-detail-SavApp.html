Projet : Savon App

Application web de dosage de savon

Journal de Bord

S1

Analyse métier : étude de l'algorithme de calcul SaF via Excel et appropriation du contexte de fabrication.

S2

Test Backend : test des points d'accès (Endpoints) de l'API Kotlin/Spring Boot avec le logiciel Bruno (CRUD).

S3

Maquettage UI/UX : conception de l'interface utilisateur pour la gestion des ingrédients et des recettes.

S4

Initialisation Angular : création du projet (CLI), routage des pages et intégration du design CSS global.

Environnement Technique

  • Frontend : Angular 17+ (TypeScript)
  • Backend : API REST Kotlin / Spring Boot
  • Tests API : Bruno (Opérations CRUD)
  • Algorithmique : Calcul de Saponification (SaF)
  • Outils : Angular CLI / VS Code
